A Guide to Tinkercad Architecture

Why would anyone in their right mind make this?

When I started architecture 2 years ago I heard about Revit, also known as the most widely used architecture software around. But when I saw that the software was over $1,000 I decided I didn't want to learn how to use it.

So instead of spending the time to learn Revit I used a web based software designed for engineering called Tinkercad.

Now if you're like me you might have heard of Tinkercad or used it before, but if you haven't, don't worry, it's very easy to pick up and learn in a few minutes. Although, this should be more than enough for you to learn how to use Tinkercad effectively.
